# Todo List (DevOps Project)

A simple Todo List application deployed using Docker and Docker Compose, with CI using GitHub Actions.

## Tech Stack
- Node.js
- Docker
- Docker Compose
- GitHub Actions (CI)

## Requirements
- Docker Desktop installed (Windows/Mac) OR Docker Engine (Linux)
- Docker Compose (Docker Desktop includes it)

## Run with Docker Compose (Recommended)

### Build + Run
```bash
docker compose up -d --build
